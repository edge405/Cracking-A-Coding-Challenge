Chocolate Feast

Maximize your chocolate consumption with this simple algorithmic solution!

Click here for more detailed problem explanation

Problem Summary

In this problem, Little Bobby loves chocolates and wants to maximize the number of chocolates he can eat. He can buy chocolates with his money, and also exchange wrappers for more chocolates.

Objective: Find out how many chocolates Bobby can eat with his given money, considering he can exchange wrappers for more chocolates.

Example

Letâ€™s break down a sample scenario:

  • Bobby has: n = 10 dollars.
  • The cost of one chocolate: c = 2.
  • Bobby can exchange wrappers: m = 5 wrappers for a free chocolate.

Solution Steps

To solve the problem, we can follow these steps:

  1. Calculate how many chocolates Bobby can initially buy with his money.
  2. Exchange wrappers for additional chocolates.
  3. Repeat the process of exchanging wrappers until Bobby can no longer exchange.

Code Solution


public static int chocolateFeast(int n, int c, int m) {
    int chocolates = n / c;
    int wrappers = chocolates;
    while (wrappers >= m) {
        int freeChocolates = wrappers / m;
        chocolates += freeChocolates;
        wrappers = wrappers % m + freeChocolates;
    }
    return chocolates;
}
