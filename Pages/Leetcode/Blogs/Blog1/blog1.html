Problem 1491: Average Salary Excluding the Minimum and Maximum Salary

In this blog post, we will dive into a simple but interesting coding problem: calculating the average salary of employees while excluding the highest and lowest salaries from the calculation. Let's break down the problem, understand the solution step by step, and provide a beginner-friendly explanation.

Problem Statement

You're given an array of unique integers, salary, where each element represents the salary of an employee. Your task is to compute the average salary of these employees, excluding the minimum and maximum salaries.

Input and Output

Input:

An array of integers representing the salaries, where:

  • The length of the array is at least 3.
  • All salary values are unique.

Output:

A double representing the average salary, excluding the highest and lowest values.

Examples

Example 1

Input: salary = [4000, 3000, 1000, 2000]

Output: 2500.00000

Explanation: The minimum salary is 1000, and the maximum salary is 4000. The average of the remaining salaries (2000 and 3000) is (2000 + 3000) / 2 = 2500.

Example 2

Input: salary = [1000, 2000, 3000]

Output: 2000.00000

Explanation: The minimum salary is 1000, and the maximum salary is 3000. The average of the remaining salary (2000) is simply 2000 / 1 = 2000.

Solution Explanation

class Solution {
    public double average(int[] salary) {
        Arrays.sort(salary); // Step 1: Sort the array
        double res = 0; // Step 2: Initialize result variable
        int s = 0; // Step 3: Initialize sum variable
        for(int i = 1; i < salary.length - 1; i++) { // Step 4: Loop through salaries, excluding min and max
            s += salary[i]; // Step 5: Sum up the salaries
        }
        return (double)s / (salary.length - 2); // Step 6: Calculate the average
    }
}
